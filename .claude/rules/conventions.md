# Conventions

- **No emojis** in code or copy unless the user explicitly asks for them.
- "Space voyage" theme: kicker text uses `◊`, vocabulary like *station / trajectoire / expéditions / comms*. Don't break it without reason.
- **French is the primary content language**; EN keys are equivalents, not direct word-for-word.
- **Profile data is real** (Dell Inspiron 7791, LinkedIn-sourced timeline, ESGI/Confluent Digital). Don't fabricate hardware, dates, or experience.
- **No Lorem ipsum** ever — the user's standing instruction. Use plausible content or stop and ask.
- **Phase-by-phase delivery** — when shipping a feature, deliver in lots, validate each, then move on. Don't dump everything silently.
